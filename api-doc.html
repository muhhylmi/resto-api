Authentication

All endpoints except /auth/register and /auth/login require a valid JWT token in the Authorization header.

Authorization: Bearer <your_jwt_token>

Token expires in 24 hours (86400 seconds). Re-login to get a new token.

Error Handling

All responses follow a consistent shape.

Success Response

{
  "success": true,
  "message": "Success",
  "data": { /* payload */ }
}

Error Response

{
  "success": false,
  "message": "Validation failed",
  "errors": {
    "name": ["Name is required"]
  }
}

HTTP Status Codes

200 OK 201 Created 204 No Content 400 Bad Request 401 Unauthorized 404 Not Found 409 Conflict 500 Internal Server Error
